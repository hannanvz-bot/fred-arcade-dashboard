## Suggested screenshots to capture:

1. **Main Dashboard** – The full arcade cabinet with S&P 500 loaded
2. **Random Play** – After pressing the orange button with a new indicator
3. **Random Compare** – Dual-indicator overlay mode
4. **Recession Toggle** – With recession shading visible
5. **Time Warp** – Zoomed to 1Y or 3Y view
6. **Combo Counter** – High combo streak (5+)
7. **Mobile View** – If testing on mobile/tablet
8. **Metrics Table** – Full table with highlighted row

## Naming convention:
- `dashboard-main.png`
- `random-play-unemployment.png`
- `compare-gdp-vs-cpi.png`
- `recession-shading.png`
- `time-warp-1y.png`
- `combo-x7.png`

## Tips for great screenshots:
- Use Chrome/Edge DevTools → Device Toolbar → 1440×900 for consistent sizing
- Capture with browser console closed for clean look
- Dark mode browser theme matches the aesthetic
- Consider recording a short GIF for the README

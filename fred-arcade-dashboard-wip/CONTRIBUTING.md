# Contributing to FRED Arcade Dashboard

Thank you for your interest in making this project more awesome! 🎮

## 🎯 How to Contribute

### Reporting Bugs
- Open an [Issue](../../issues) with the label `bug`
- Include: Python version, OS, browser, and steps to reproduce
- Attach screenshots if UI-related

### Suggesting Features
- Open an [Issue](../../issues) with the label `enhancement`
- Describe the feature and why it fits the "arcade" theme
- Mockups or sketches are welcome!

### Pull Requests
1. Fork the repository
2. Create a feature branch: `git checkout -b feature/amazing-idea`
3. Commit your changes: `git commit -m 'Add amazing idea'`
4. Push to the branch: `git push origin feature/amazing-idea`
5. Open a Pull Request

### Code Style
- Follow PEP 8 for Python code
- Use meaningful variable names (even in retro style)
- Comment complex FRED API logic
- Keep the arcade aesthetic consistent (neon colors, pixel fonts)

### Commit Messages
Use emoji prefixes for fun:
- 🎮 `🎮 feat:` new feature
- 🐛 `🐛 fix:` bug fix
- 📚 `📚 docs:` documentation
- 🎨 `🎨 style:` UI/CSS changes
- ⚡ `⚡ perf:` performance improvement

## 🏆 Recognition

Contributors will be added to the README credits section. Top contributors get a special spot on the "Arcade Hall of Fame" (when we build it 😉).

## 📜 Code of Conduct

Be respectful, constructive, and remember: **we're all here because economics should be fun.**
